Introduction

A 2.5 months industrial training program on full stack web development was conducted by ShapeAI to equip interns with the unique skills they need to build database-backed APIs and web applications with the proper industry-level front-end as well as scale up the deployment for the millions.


A graduate of this program will be able to design and build databases for software applications, create and deploy database-backed web APIs, and secure and manage user authentication and access control for an application backend. They will be able to build production level frontend according to the industrial standards. Students will also learn how to deploy a the entire web application to the cloud using Docker and Kubernetes and how to maintain and scale them.
